Hello world

This server is running on a Raspberry pi using Docker with container running Postgres Databasebr
Serve using Nodejs, Express, Prisma for mapper and exposing to the world using CloudFlared tunnel

Technology in used

  • Node.js
  • TypeScript
  • Express.js
  • Prisma
  • PosgresQl

API's

User API

  • List all User
    https://postgres.nozky.ml/user

  • Find User by ID
    https://postgres.nozky.ml/user/id?id="userId"

  • Add user
    POST: https://postgres.nozky.ml/user

    body: { "id": "1", "name": "YourName", "email": "yourName@mail.com", "age": 0 }

  • Update user
    POST: https://postgres.nozky.ml/user/Update

    body: { "id": "1", "name": "Updated name", "email": "Updated yourName@mail.com", "age": 0 }

  • Detele user
    POST: https://postgres.nozky.ml/user/Delete

    body: { "id": "userId" }

This is just a test to see if Raspberry pi 4 will be able to handle small scale businesses.
