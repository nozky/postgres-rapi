Hello world!

This server is running on a Raspberry pi in my house using Docker and container running Postgres Database
Serve using Nodejs, Express, Prisma and exposing to the world using CloudFlared tunnel

Technology in used

  • Nodejs
  • TypeScript
  • Express
  • Prisma
  • Posgres

API's

User API

  • List all User
    GET: https://postgres.nozky.ml/user

  • Find User by ID
    GET: https://postgres.nozky.ml/user/id?id="userId"

  • Find User that contain name
    GET: https://postgres.nozky.ml/user/user/name?name="userName"

  • Add user
    POST: https://postgres.nozky.ml/user

    body: { "id": "1", "name": "YourName", "email": "yourName@mail.com", "age": 0 }

  • Update user
    POST: https://postgres.nozky.ml/user/Update

    body: { "id": "1", "name": "Updated name", "email": "Updated yourName@mail.com", "age": 0 }

  • Detele user
    POST: https://postgres.nozky.ml/user/Delete

    body: { "id": "userId" }

Post API

  • List all Post
    GET: https://postgres.nozky.ml/post/all

  • Add Post
    POST: https://postgres.nozky.ml/post

    body: { "title": "Your unique title", "body": "post body", "authorId": "Existing authorId" }

  • Update Post
    PUT: https://postgres.nozky.ml/post/

    body: { "id": "post id" "title": "Your unique title", "body": "post body", }

  • Delete Post
    DELETE: https://postgres.nozky.ml/post/delete/id?id=postId

    body: { "id": "post id" "title": "Your unique title", "body": "post body", }

  • Delete all Post from author
    DELETE: https://postgres.nozky.ml/post/deleteall

    body: { "authorId": "authorId" }

  • Find all Post from Author
    GET: https://postgres.nozky.ml/post/authorId

    body: { "authorId": "author id" }

  • Find single Post
    GET: https://postgres.nozky.ml/post/id

    body: { "id": "post id" }

This is just a test to see if Raspberry pi 4 will be able to handle small scale businesses.
